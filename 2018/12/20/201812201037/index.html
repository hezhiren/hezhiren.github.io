手机号脱敏处理

1.使用正则表达式

 // 手机号码前三后四脱敏
public static String mobileEncrypt(String mobile) {
    if ((mobile.length() != 11)) {
        return mobile;
    }
    return mobile.replaceAll("(\\d{3})\\d{4}(\\d{4})", "$1****$2");
}

1.身份证脱敏处理

//身份证生日四位脱敏
public static String IdCardEncript(String cardNo) {
    if(cardNo == null || (cardNo.length() != 18)) {
        return cardNo;
    }
    return cardNo.replaceAll("(\\d{10})\\d{4}(\\d{4})", "$1****$2");
}
